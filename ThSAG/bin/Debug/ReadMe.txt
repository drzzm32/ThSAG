■１．声明
	这是一个AVG+STG的简易游戏引擎，基于VB .NET和DxLib。
	资源加载可能重复。
	在场景n定义背景音乐，要跳转到场景n+1才会播放。
	点击程序左上角“THSAG”图标可以切换为开发模式，并激活Script按键。
	更多信息，请点击Script Help。
	「Script.txt」为一范例脚本（资源配置文件）。


■２．安装方式
	借用神主的一句话，「既然是微软不支持的东西我们也不能支持」
	因此我用了.NET Framework 4.5

	.NET Framework 4.0
	（若是Windows7这个就不用装了）

	.NET Framework 4.0
	可以留着，用处很大。
